# SMALLSH README

To compile and run the program successfully, please follow these steps:

1. Navigate to the directory containing mortonwi_program3.zip.
2. Unzip the file.
3. Run the following command in a compatible terminal in the same directory to compile the program:
    gcc --std=gnu99 -o smallsh main.c
4. The executable file should now be ready to use!

# FEATURES

* File navigation (ls, cd)
* Executing background operations (fork)
* Standard command line interface
